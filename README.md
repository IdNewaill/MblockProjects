# Mblock Projects
[Mblock](https://mblock.cc/) est une plateforme comme [Scratch](https://scratch.mit.edu) qui permet de coder avec des blocs ou ici du Python mais permet en plus de programmer sur des appareils comme des Arduinos, ESP32 ou tout autres appareils.

Ce répertoire contiendra essentiellement des programmes Python fait pour [Cyberpi](https://youtu.be/s004xBcasJU) (qui est en vérité un assemblage avec un ESP32).
